Задача

Напишите на JavaScript функцию, которая выводит список всех чисел, которые равны сумме факториалов своих цифр.

Пример такого числа:
4! + 0! + 5! + 8! + 5! = 40585.

Решение

Такие числа называются факторионами. Для решения задачи необходимо определить границы поиска.

Сумма факториалов не может превышать:
9! × d, где d – количество цифр.

Число быть меньше:
10d-1.

Должно выполняться условие:
10d-1 ≤ n ≤ 9! × d, где n – факторион.

Следовательно, факторионы состоят не более, чем из 7 цифр. Верхняя граница — 2540160.
